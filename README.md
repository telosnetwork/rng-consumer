# Telos RNG Consumer

## Setup

To begin, navigate to the project directory: `rng-consumer/`

    mkdir build && mkdir build/rng-consumer

    chmod +x build.sh

    chmod +x deploy.sh

## Build

    ./build.sh rng-consumer

## Deploy

    ./deploy.sh rng-consumer { account-name } { mainnet | testnet | local }
